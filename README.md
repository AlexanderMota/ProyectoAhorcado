# Juego del Ahorcado - Aplicación WPF en C#

Este proyecto es una aplicación de escritorio que simula el clásico juego del ahorcado, desarrollada utilizando **Windows Presentation Foundation (WPF)** en C#. Fue creada como parte de mis estudios en el *Ciclo Superior de Desarrollo de Aplicaciones Multiplataforma (DAM)*. El juego selecciona palabras aleatorias de un archivo `.txt` y desafía al jugador a adivinarlas antes de que el personaje sea "ahorcado".

## Características principales

- **Interfaz gráfica con WPF:** Diseño intuitivo y visualmente atractivo que mejora la experiencia del usuario.
- **Selección aleatoria de palabras:** Las palabras a adivinar se eligen automáticamente de un archivo de texto incluido en la aplicación.
- **Gestión de intentos:** Control de intentos fallidos y visualización del progreso del jugador.
- **Código estructurado:** Implementación en C# siguiendo buenas prácticas de programación y separación de responsabilidades.

## Estructura del proyecto

### Archivo de palabras
- Un archivo `.txt` que contiene una lista de palabras utilizadas aleatoriamente en cada partida.

### Lógica del juego
- Clases y métodos en C# diseñados para gestionar las reglas del ahorcado, como verificar letras, manejar intentos y mostrar el estado del juego.

### Interfaz de usuario
- Interfaz gráfica construida con WPF, proporcionando una experiencia de usuario interactiva y amigable.

## Futuras funcionalidades

- **Sistema de puntuación:** Evaluar el desempeño del jugador.
- **Modo multijugador:** Integración de un modo local o en línea.
- **Personalización:** Opciones para editar las palabras directamente desde la aplicación.
